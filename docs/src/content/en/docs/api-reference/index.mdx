# API Reference

This section provides detailed documentation on all public APIs offered by the `SAuthBase` SDK.  
Each method, function, and type definition includes comprehensive information such as purpose, usage, return values, examples, and error codes.

  - Fetching Scratch user information

  - Token verification, generation, and decoding
- Type Definitions (`APIResult`, `userInfoProps`, etc.)
