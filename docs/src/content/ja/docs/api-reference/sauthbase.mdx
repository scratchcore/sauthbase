# sauthbase クラス

`sauthbase` は SDK のメインクラスであり、すべての認証・暗号処理機能のインターフェースを提供します。

## 概要

- シングルトンで動作
- `init()` で初期化、`use()` で使用
- `secretKey` および `redirect_url` を引数に初期化

## 初期化

```ts filename="TypeScript" showLineNumbers twoslash
// @noErrors
import { sauthbase } from "sauthbase";
// ---cut---
const sab = sauthbase.init({ secretKey: "...", redirect_url: "..." });
```

## 使用

```ts filename="TypeScript" showLineNumbers twoslash
// @noErrors
import { sauthbase } from "sauthbase";
// ---cut---
const sab = sauthbase.use();
```
